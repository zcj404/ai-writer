========================================
墨笔AI写作助手 - 部署说明
========================================

【前提条件】
- 阿里云 ECS（Ubuntu 20.04/22.04）
- 安全组开放端口：80（HTTP）、22（SSH）

========================================
零、推送代码到 GitHub（首次）
========================================

1. 在 GitHub 创建新仓库（不勾选 Initialize README）

2. 本地执行（Git Bash）：
   cd G:/Project
   git remote add origin https://github.com/你的用户名/仓库名.git
   git push -u origin master

3. 确认 .gitignore 包含以下内容（避免泄露敏感信息）：
   server/.env
   server/data.db
   node_modules/
   client/build/
   .claude/

========================================
一、服务器初始化（首次部署）
========================================

1. SSH 登录服务器
   ssh root@你的服务器IP

2. 安装 Node.js 20
   curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
   apt-get install -y nodejs

3. 安装 PM2 和 Nginx
   npm install -g pm2
   apt-get install -y nginx

4. 从 GitHub 拉取代码
   cd /opt
   git clone https://github.com/你的用户名/你的仓库名.git novel-ai
   cd /opt/novel-ai

5. 构建前端
   cd /opt/novel-ai/client
   npm install
   npm run build
   # 构建产物在 client/build，后端会自动托管

6. 安装后端依赖
   cd /opt/novel-ai/server
   npm install --production

7. 创建后端环境变量文件
   cat > /opt/novel-ai/server/.env << EOF
   DASHSCOPE_API_KEY=你的DashScope密钥
   JWT_SECRET=随机字符串（可用 openssl rand -hex 32 生成）
   PORT=3001
   NODE_ENV=production
   EOF

8. 配置 Nginx
   cp /opt/novel-ai/nginx.conf /etc/nginx/sites-available/novel-ai
   ln -sf /etc/nginx/sites-available/novel-ai /etc/nginx/sites-enabled/novel-ai
   rm -f /etc/nginx/sites-enabled/default
   # 编辑 nginx.conf，将 your-domain.com 改为你的服务器IP或域名
   nano /etc/nginx/sites-available/novel-ai
   nginx -t && systemctl reload nginx

9. 启动应用
   cd /opt/novel-ai
   pm2 start ecosystem.config.js
   pm2 save
   pm2 startup | tail -1 | bash

10. 访问
    http://你的服务器IP

========================================
二、上传本地数据库（迁移测试数据）
========================================

如需将本地测试账号和数据一起迁移到服务器：

在本地 Windows 执行（用 Git Bash）：
   scp G:/Project/server/data.db root@你的服务器IP:/opt/novel-ai/server/data.db

上传后重启应用：
   pm2 restart novel-ai

========================================
三、后续更新代码
========================================

SSH 登录服务器后执行：

   cd /opt/novel-ai
   git pull

   # 如果前端有改动，重新构建
   cd client && npm install && npm run build && cd ..

   # 如果后端依赖有改动
   cd server && npm install --production && cd ..

   # 重启后端
   pm2 restart novel-ai

========================================
四、常用 PM2 命令
========================================

pm2 status              # 查看运行状态
pm2 logs novel-ai       # 查看日志
pm2 restart novel-ai    # 重启
pm2 stop novel-ai       # 停止

========================================
五、注意事项
========================================

1. .env 文件不要提交到 GitHub，确保 .gitignore 包含：
   server/.env
   server/data.db（可选，如不想把数据库提交到 git）

2. DASHSCOPE_API_KEY 在阿里云百炼控制台获取：
   https://bailian.console.aliyun.com/

3. 如果 better-sqlite3 安装失败，需要安装编译工具：
   apt-get install -y python3 make g++
   cd /opt/novel-ai/server && npm install --production
